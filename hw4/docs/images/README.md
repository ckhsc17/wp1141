## 🎨 圖片規範

### 截圖規範
- **格式**：PNG (推薦) 或 JPG
- **解析度**：1920x1080 或更高
- **壓縮**：適度壓縮以平衡品質和檔案大小
- **命名**：使用描述性的英文名稱，用連字符分隔

### Logo 規範
- **格式**：PNG (支援透明背景)
- **尺寸**：建議 512x512 或更高
- **背景**：透明背景優先

## 📝 使用方式

在 README.md 中引用圖片：

```markdown
![應用首頁](docs/images/screenshots/homepage.png)

<img src="docs/images/logos/logo.png" alt="尋寶地圖 Logo" width="200">
```
