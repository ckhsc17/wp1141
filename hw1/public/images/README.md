# 圖片上傳指南

## 個人照片位置
請將你的個人照片放在這個目錄下，並命名為：
- `profile.jpg` 或 `profile.png`

## 建議的圖片規格
- **格式**：JPG、PNG 或 WebP
- **尺寸**：至少 400x400 像素，建議 800x800 像素（正方形）
- **檔案大小**：盡量控制在 500KB 以下以優化載入速度
- **風格**：專業照片，建議使用高品質的頭像照片

## 如何更換圖片
1. 將你的照片放到 `public/images/` 目錄下
2. 重新命名為 `profile.jpg` 或 `profile.png`
3. 如果需要使用不同的檔案名，請修改 `src/data/mockData.ts` 中的 `profileImage` 路徑

當前設定的圖片路徑：`/images/profile.jpg`
