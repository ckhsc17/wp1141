# Booboo 小幽使用指南

## 簡介

Booboo 小幽是一個個人生活記錄與 AI 助手 LINE Bot，幫助你整理靈感、知識、記憶，並提供個人化的回饋與推薦。
可以直接以自然語言和小幽對話，讓你可以馬上上手！

## 功能說明

### 1. 待辦事項 (Todo)

#### 新增待辦
- 「我要吃飯、取貨、寫作業」
- 「待辦：買菜」
- 「1. 吃飯 2. 取貨 3. 寫作業」

#### 更新待辦狀態
- 「我寫完作業了！」
- 「作業完成了」
- 「取消吃飯」

#### 查詢待辦
- 「我上禮拜做了哪些事？」
- 「吃了什麼？」
- 「查看待辦」
- 「12/25 的待辦」
- 「我下禮拜有哪些待辦事項」

### 2. 資訊連結 (Link)

分享連結時，小幽會自動分析連結內容並儲存：
- 「https://example.com/article」
- 「這個文章很有趣 https://example.com」

### 3. 靈感 (Insight)

記錄文學上、生活上的頓悟和啟發：
- 「今天突然理解了一個人生道理」
- 「這本書給我的啟發是...」
- 「突然想到...」

### 4. 知識 (Knowledge)

記錄資訊技術、學術、常識等：
- 「React Hooks 的用法」
- 「量子力學的基本概念」
- 「學到一個新技巧...」

### 5. 記憶 (Memory)

記錄個人經驗、日記、與某人的對話等：
- 「今天跟朋友聊到...」
- 「記得上次去...」
- 「今天發生了一件有趣的事...」

### 6. 音樂 (Music)

記錄想練習的歌曲：
- 「在 solo」
- 「陶喆 蝴蝶」
- 「1. 館青 青少年觀察日記」
- 「2. 方大同 love song」

### 7. 生活活動 (Life)

記錄展覽、電影、想從事的活動：
- 「小巨蛋溜冰！」
- 「左撇子女孩金馬獎」
- 「人也太好了吧展」

### 8. 回饋請求 (Feedback)

要求生活建議或回饋：
- 「給我一些生活建議」
- 「幫我分析時間管理」
- 「我需要一些回饋」

### 9. 推薦請求 (Recommendation)

要求推薦內容：
- 「推薦一些技術文章」
- 「有什麼好聽的音樂？」
- 「推薦一些展覽」

### 10. 對話紀錄查詢 (Chat History)

查詢過往對話：
- 「我有沒有跟 XXX 聊到過 XXX？」
- 「我上禮拜說了什麼？」
- 「之前提到的作業是什麼？」

### 11. 一般聊天 (Other)

與小幽自由對話：
- 「你好」
- 「今天天氣如何？」
- 「給我一些鼓勵」

## Tags 使用方式

所有儲存的內容都會自動標記 tags（標籤），用於後續的搜尋與推薦：

- **insight**: 靈感相關
- **knowledge**: 知識相關
- **memory**: 記憶相關
- **music**: 音樂相關
- **life**: 生活活動相關
- **link**: 連結相關
- 其他自訂標籤（例如：technology, react, life, time-management 等）

## RAG 查詢運作原理

當你請求推薦或回饋時，小幽會：

1. **提取標籤**：從你的查詢中提取相關 tags
2. **搜尋相關內容**：根據 tags 從你的儲存內容中搜尋相關項目
3. **生成回應**：基於搜尋結果，使用 AI 生成個人化的推薦或回饋

這確保了小幽的回應是基於你過去的記錄，提供更貼近你需求的建議。

標籤：insight, life, philosophy
```

小幽：已儲存知識
標籤：knowledge, technology, react
```

### 範例 3：查詢待辦
```
你：我上禮拜做了哪些事？
小幽：[顯示待辦事項列表]
```

### 範例 4：請求推薦
```
你：推薦一些技術文章
小幽：[基於你過去的技術相關記錄，提供個人化推薦]
```

### 範例 5：請求回饋
```
你：給我一些生活建議
小幽：[基於你過去的生活記錄，提供個人化回饋]
```
